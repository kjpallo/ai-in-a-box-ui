# Charlemagne Motion and Force Knowledge Pack

Source: user-provided "Packet KEY Motion and Forces CP.pdf" (Motion and Force unit).

This pack is organized for Charlemagne ingestion and Codex review.

Files:
- motion_force_vocabulary.csv: student-friendly vocabulary, misconception notes, examples, source page ranges.
- motion_force_formulas.csv: formulas, rearrangements, trigger phrases, and solution steps.
- motion_force_concepts.csv: concept-level learning targets and router support notes.
- motion_force_graph_patterns.csv: distance-time and velocity-time graph interpretation patterns.
- motion_force_problem_bank.csv: original/paraphrased practice problems aligned to the packet skills. These are not copied verbatim from the publisher worksheet.
- motion_force_smoke_tests.csv: quick router test cases to add to scripts/test-router.js or a new physics smoke test.

Suggested repo placement:
lib/knowledge/physics/motion-force/
scripts/import-motion-force-knowledge.js
scripts/test-motion-force-knowledge.js

Copyright note:
The problem rows are new paraphrased/generated items based on the skills in the packet, rather than direct copies of the worksheet wording.
